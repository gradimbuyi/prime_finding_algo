# Prime Finder Documentation

A simple program to find every prime number within the range of $0 \rightarrow 10^{10}$. It makess use of multithreading to accomplish this.

## Compiling the code:

```
g++ -o code code.cpp
```

## Running the code:

```
./code
```

### Output

Should produce a primes.txt file that prints out the execution time, the prime count, the sum of primes,
and the top 10 prime numbers.
